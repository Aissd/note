### 全局安装
> npm install -g typescript

### 校验
> tsc -v
> tsc作用：负责将ts代码转为浏览器和nodejs识别的js代码

### 命令行运行ts
```
tsc index.ts
```

### 自动编译
* 运行 tsc --init，创建tsconfig.json

* 修改tsconfig.json文件，设置js文件夹：'outDir': './js/'
