```
// 查看全局配置
git config --list

// 修改
git config --global user.name 'zouyushen'

git config --global user.email '335498421@qq.com'

// 添加
git add .

// 提交到本地仓库
git commit -m "Initial commit"

// 推送到远程仓库
git push -u origin master

快速删除node_modules目录
cnpm install rimraf -g
rimraf node_modules

切换到指定版本
git reset --hard 版本号

回退到上一个版本
git reset --hard HEAD^

查看当前分支
git branch

查看当前git项目地址
git remove -v

建立一个空文件
touch index.js

建立一个空文件夹
mkdir demo

初始化一个git仓库
git init

查看当前目录所有文件
ls

查看当前目录路径
pwd

添加别名
git config --global alias.a add（添加add的别名 a）
因此可以 git a .
```
