数组中的常用方法：

```
按照四个维度记忆：
1）方法的作用
2）方法的参数
3）方法的返回值
4）原有数组是否改变
```

1、push

```
1）作用：向数组末尾追加新的内容
2）参数：追加的内容（可以是一个，也可是多个）
3）返回值：新增后数组的长度
4）原有数组改变
```

2、pop

```
1）作用：删除数组最后一项
2）参数：无
3）返回：被删除的那一项内容
4）原有数组改变
```

3、shift

```
1）作用：删除数组中的第一项
2）参数：无
3）返回：被删除的那一项内容
4）原有数组改变
第一项被删后，原有后面每一项的索引都要向前减一
```

4、unshift

```
1）作用：向数组开始位置追加新内容
2）参数：要新增的内容
3）返回：新增后数组的长度
4）原有数组改变
```

5、splice

```
1）作用：
基于splice可以对数组进行很多的操作：
    1.1）删除指定位置的内容；
    1.2）向数组指定位置增加内容；
    1.3）还可以修改指定位置的信息
2）参数：
	2.1）删除：ary.splice(n, m);
	从索引n开始，删除m个内容，把删除的部分以一个新数组返回，原有数组改变；
	如果不指定m，或者删除的个数大于最大长度，都是删除到数组的末尾。
	2.2）新增：ary.splice(n, 0, x, ...);
	从索引n开始删除0项（没删除，返回[]），把x或者更多需要插入的内容存放到数组中索引n的前面
	2.3）修改：ary.splice(n, m, x, ...)
	修改的原理就是把原有内容删除掉，然后用新的内容替换这部分信息即可
3）返回：把删除的部分以一个新的数组返回
4）原有数组改变
```

6、slice

```
1）作用：在一个数组中，按照条件查找出其中的部分内容
2）参数：两个参数(n/m) 从索引n开始，查找到索引m，但是不包含m
3）返回：以一个新数组存储查找的内容
4）原有数组不会变

ary.slice(0); // 从索引0开始，查找到结束，相当于克隆一个新数组并返回
ary.slice(); // 与上述传0同理
ary.slice(2); // 从索引2开始，查找到结束

let ary = [1,2,3,4,5];
ary.slice(-3, -1); // 可以理解为，总长度加上负数索引，即等同于ary.slice(2, 4); // [3, 4]
```

7、concat

```
1）作用：实现多个数组（或者值）的拼装
2）参数：数组或者值
3）返回：拼接后的新数组
4）原有数组不会变
```

8、toString

```
1）作用：把数组转换为字符串
2）参数：无
3）返回：数组中的每一项用逗号分隔的字符串
4）原有数组不会变
```

9、join

```
1）作用：和toString类似，也是把数组转换为字符串，但是我们可以设置变为字符串后每一项之间的连接符
2）参数：指定的连接符
3）返回：字符串
4）原有数组不会变

基于join我们可以实现数组中每一项的求和功能。
1、基于json，使用+作为分隔符，先把数组变为每一项相加的字符串；
2、基于eval，把字符串变为js表达式执行，得到的结果就是数组中每一项累加的和
```

10、reverse

```
1）作用：把数组倒过来排列
2）参数：无
3）返回：排列后的新数组
4）原有数组改变
```

11、sort

```
1）作用：给数组排序
2）参数：无/函数
3）返回：返回排序号的新数组
4）原有数组改变

sort在不传参数的情况下，只能处理10以内的数字排序。
真实项目中，基于sort排序，我们都需要传递参数

ary.sort(function(a, b) {
	return a-b; // 升序
	// return b-a; // 降序
});
```

12、indexOf/lastIndexOf

```
这两个方法不兼容IE低版本浏览器（IE6~IE8）
1）作用：检测当前值在数组中第一次或者最后一次出现的索引
2）参数：要检测的值
3）返回：索引
4）原有数组不会变

基于indexOf，如果数组中右这一项返回一个大于等于0的索引，如果没有这一项，返回的索引为-1
```

13、find/findIndex

```
这两个方法不兼容IE浏览器（包括IE11）
find()
1）返回符合函数条件的第一个数组元素，且之后的遍历不再执行；
2）没有找到符合函数条件时，返回undefined；
3）空数组遍历，函数是不会执行的

findIndex()
1）返回符合函数条件的第一个数组元素索引位置，且之后的遍历不再执行；
2）没有找到符合函数条件时，返回-1；
3）空数组遍历，函数是不会执行的
```

var obj = {};
for(var i = 0; i < ary.length; i++) {
	var item = ary[i];
	// 存储之前需要做判断：如果对象中已经存在这个属性了，说明当前item在之前出现过，也就是当前项重复了，就把当前项删掉
	if(typeof obj[item] !== 'undefined') {
		// 1、把数组最后一项的结果获取到，替换当前项内容
		// 2、再把数组最后一项删除
		// [12,23,34,56]，想要删除23
		// 先让56替换23 [12,56,34,56]
		// 再把最后一项删除 [12,56,34]
		ary[i] = ary[ary.length - 1];
		ary.length--;
		i--;
		continue;
	}
	// 把这一项作为对象的属性名和属性值存储进去
	obj[item] = item; // 
}
