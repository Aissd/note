### 一元加法

```
+10; // 10 - 对数字类型无作用
+'10'; // 10 - 对字符串会转换成数字
+'a'; // NaN - 与Number()转换一致，遇到非有效数字，直接返回NaN
+null; // 0
+undefined; // NaN
```

### 一元减法

```
-10; // -10
-'10'; // -10
-'a'; // NaN
-null; // -0
-undefined; // NaN

```
### 指数操作符
```
Math.pow(3, 2); // 9
3 ** 2; // 9
// 开方
Math.pow(16, 0.5); // 4 
16 ** 0.5; // 4

let squared = 3;
squared **= 2; // 9

let sqrt = 16;
sqrt **= 0.5; // 4

```
