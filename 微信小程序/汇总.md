10、页面间传递数据

```
A页面跳B页面
A：
wx.navigateTo({
	url: '../A/A?id=1'
});
// 另一种跳转方式
<navigator url="../A/A?id=1">点击跳转</navigator>
// 重定向
<navigator url="../A/A?id=1" redirect>点击跳转</navigator>
B：
Page({
	onLoad: function(options) {
		console.log(options); // { id: 1 }
	}
});
```
